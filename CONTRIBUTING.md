# 기여 가이드

연대지능 공동위키에 문서를 기여하는 방법입니다. GitHub가 처음이어도 자기 AI(Claude Code 등)와 함께 따라 하면 됩니다.

## 1. Fork (내 계정으로 복제)

이 레포 오른쪽 위 **Fork** 버튼을 누릅니다. 그러면 `내계정/solidarity-intelligence-wiki`가 생깁니다.

## 2. 문서 작성

- `문서/` 아래 알맞은 분류 폴더에 `.md` 파일 1편을 만듭니다.
- 파일 맨 위에 **frontmatter**(제목·작성자)를 넣습니다. [`문서/_템플릿.md`](문서/_템플릿.md)를 복사해서 시작하세요.

```markdown
---
title: 문서 제목
author: 내-GitHub-핸들
date: 2026-07-03
---

여기에 본문...
```

## 3. Pull Request 제출

- 바뀐 내용을 커밋하고, 자기 Fork에서 이 레포로 **Pull Request**를 엽니다.
- PR 제목: `[문서] 제목` / 설명: 어떤 문서를 왜 넣었는지 한두 줄.

## 4. 검토와 머지

- **공동 편집자**가 내용을 읽고, 공동위키에 맞으면 **머지**합니다.
- 형식 문제(빌드 깨짐·비밀키·개인정보)는 사전 자동 검토가 먼저 걸러줍니다. 편집자는 **주제가 맞는지 내용만** 봅니다.
- 수정 요청이 달리면, 그건 더 좋은 문서를 만드는 과정입니다. 반영해서 다시 올리면 됩니다.

## 하지 말 것

- 🔴 **개인정보(이름·연락처·주소)·비밀키·비밀번호 올리지 않기.** 공개 레포입니다.
- 🔴 다른 사람 글을 통째로 복사해 오지 않기 (출처를 밝히고 인용).
- 🔴 빈 PR(문서 없이) 제출하지 않기.
