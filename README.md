# 연대지능 공동위키 (Solidarity Intelligence Wiki)

> 연대지능 활동가와 학습자가 **함께 만드는 공개 지식베이스**입니다.

## 이 위키는 무엇인가

연대지능 AI활동가 양성과정 참여자와 활동가들이, 각자의 분야에서 쌓은 지식을 문서로 기여해 함께 축적하는 **공동 위키**입니다.

- 각자 자기 사이트에 세우는 **개인 위키**와 달리, 여기는 **모두가 함께 편집**하는 공동 공간입니다.
- 누구나 문서를 제안(PR)할 수 있고, **공동 편집자**가 내용을 검토한 뒤 반영(머지)합니다.

## 기여하는 방법 (Fork → 문서 → PR)

1. 오른쪽 위 **Fork** 버튼으로 이 레포를 자기 계정에 복제합니다.
2. `문서/` 아래 알맞은 분류 폴더에 **마크다운 문서 1편**을 작성합니다. (형식은 [`문서/_템플릿.md`](문서/_템플릿.md) 참고)
3. **Pull Request(PR)** 를 제출합니다.
4. **공동 편집자**가 내용을 검토하고 **머지**하면 공동위키에 반영됩니다.

자세한 절차는 [`CONTRIBUTING.md`](CONTRIBUTING.md)를 보세요.

## 분류 (`문서/`)

| 폴더 | 담는 내용 |
|------|-----------|
| `사회운동/` | 운동 이론·사례·역사 |
| `실천기술/` | 도구·방법·워크플로우 |
| `용어사전/` | 개념 정의 |
| `분야지식/` | 각자 전문 분야의 지식 |

폴더가 없으면 새로 만들어도 됩니다. (PR 설명에 어디에 왜 넣었는지 한 줄 적어주세요.)

## 공동 편집자

제출된 PR을 검토·머지하는 역할입니다.
